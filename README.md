# ⚡ Flash Keychain Design

> **Developed by:** Meshal Al Mehmady

---

## 📌 Project Overview

This project showcases the design of a **Flash-inspired keychain** created using **Onshape** for 3D printing.

The design was recreated by estimating the dimensions of a reference image, adding a **4 mm** keyring hole, extruding the model to a thickness of **2 mm**, and exporting it as an **STL** file.

## 🖨️ STL Export from Onshape

The final keychain model was exported from **Onshape** as an **STL file** for 3D printing.

### Export Steps:

1. Open the final design in **Onshape Part Studio**.

2. Right-click on the final part or the Part Studio tab.

3. Select **Export** from the menu.

4. Configure the export settings:
   - **Format:** STL
   - **Units:** Millimeter (mm)
   - **STL File Format:** Binary
   - **Resolution:** Fine

5. Click **Export** to generate the STL file.

6. The exported file (**Flash-Keychain.stl**) is ready for 3D printing and fabrication.
